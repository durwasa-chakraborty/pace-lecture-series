#### **Talk**
- **Topic:** What is in an Inductive Type? A look in with Lean

- **Description:** Inductive types are ubiquitous in functional programming as a very powerful programming paradigm. Also known as sum types, these are used to define types with multiple constructors, each of possibly different arguments and types themselves. However, these same types lend themselves to defining recursive functions, which come with nuances of their own - how do we show termination, for example? These also lend themselves well to the principle of mathematical induction, which several mathematics proofs are based on. This talk will look into these aspects using the theorem proving language Lean.
